# Omni Twin

### Usage

Omni Twin is a Unity project.

1. Download [Unity](https://unity.com/download)
2. Open the project in Unity.

## Flood Simulation

The flood simulation demo is done using compute shaders, utilizing GPGPU for accelerated computing.
